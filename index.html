CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use Inline CSS to style element directly in HTML
  • Use Internal Style Sheet to embed an entire stylesheet durectly in the HTML file, using style or head elements
  • External CSS Stylesheet allows developers to keep their CSS rules in seperate files
  • Declaration contains two important components: CSS property and value of property
